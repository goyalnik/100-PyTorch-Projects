## 📌 Description
Sentiment analysis using Bag-of-Words vectors and a feedforward neural network.

## 🎯 Goal
Classify movie reviews as Positive or Negative.

## 🛠️ Tech Stack
- Python 3.x
- PyTorch
- scikit-learn

## 🚀 How to Run
pip install -r requirements.txt
python train.py
python predict.py

## 📊 Expected Output
Epoch 20, Loss: 0.4821
...
Epoch 100, Loss: 0.1023

Review: "This movie was absolutely wonderful"
Sentiment: Positive 😊 (Score: 0.8923)
