# Project 1 - Linear Regression From Scratch using PyTorch

This project demonstrates a simple linear regression model using PyTorch.

## Features

- Linear Regression
- PyTorch Neural Network
- SGD Optimizer
- MSE Loss Function
- Model Saving and Loading
- Prediction Script

## Installation

```bash
pip install -r requirements.txt
```

## Run Training

```bash
python train.py
```
