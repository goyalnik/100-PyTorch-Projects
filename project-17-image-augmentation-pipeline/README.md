## 📌 Description
Image augmentation pipeline using torchvision.transforms on CIFAR-10 dataset.

## 🎯 Augmentations Used
- RandomHorizontalFlip
- RandomRotation(20)
- ColorJitter
- RandomResizedCrop

## 🛠️ Tech Stack
- Python 3.x
- PyTorch
- torchvision
- matplotlib

## 🚀 How to Run
pip install -r requirements.txt
python train.py
python predict.py
