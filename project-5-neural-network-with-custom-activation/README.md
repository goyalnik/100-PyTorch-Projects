# Project 5 - Neural Network with Custom Activation

## 📌 Description
Feedforward neural network using a custom Swish activation function.

## 🎯 Goal
Learn a noisy sine wave using custom activation:
Swish(x) = x * sigmoid(x)

## 🛠️ Tech Stack
- Python 3.x
- PyTorch

## 🚀 How to Run
pip install -r requirements.txt
python train.py
python predict.py

## 📊 Expected Output
Epoch 40, Loss: 0.0312
Epoch 80, Loss: 0.0187
Epoch 120, Loss: 0.0134
Epoch 160, Loss: 0.0098
Epoch 200, Loss: 0.0076
