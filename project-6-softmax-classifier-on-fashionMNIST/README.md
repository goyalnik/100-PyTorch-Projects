# Project 6 - Softmax Classifier on FashionMNIST

## 📌 Description
Softmax classifier to categorize clothing items from FashionMNIST dataset into 10 classes.

## 🎯 Classes
T-shirt, Trouser, Pullover, Dress, Coat, Sandal, Shirt, Sneaker, Bag, Ankle boot

## 🛠️ Tech Stack
- Python 3.x
- PyTorch
- torchvision

## 🚀 How to Run
pip install -r requirements.txt
python train.py
python predict.py

## 📊 Expected Output
Epoch 5, Loss: 0.4821
Test Accuracy: 83.45%
