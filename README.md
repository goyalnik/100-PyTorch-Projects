## 🚀 About This Repository

This repository contains **100 Deep Learning projects** built from scratch using PyTorch.  
Each project is self-contained with:
- ✅ Clean, well-commented code
- ✅ `train.py` — model training
- ✅ `predict.py` — inference & predictions
- ✅ `requirements.txt` — dependencies
- ✅ `README.md` — project explanation

> 💡 **Perfect for:** Students, developers building portfolios, and researchers exploring PyTorch.

## ⚙️ How to Run Any Project

```bash
# 1. Clone the repo
git clone https://github.com/goyalnik/100-PyTorch-Projects.git
cd 100-PyTorch-Projects

# 2. Go to any project folder
cd project-1-linear-regression

# 3. Create virtual environment
python -m venv venv
venv\Scripts\activate   # Windows
source venv/bin/activate  # Mac/Linux

# 4. Install dependencies
pip install -r requirements.txt

# 5. Train the model
python train.py

# 6. Run predictions
python predict.py
```
