# Project 2 - Logistic Regression Classifier

## 📌 Description
Binary classifier using logistic regression in PyTorch.
Classifies points in 2D space using sigmoid activation and binary cross-entropy loss.

## 🎯 Goal
Predict binary labels (0 or 1):

y = sigmoid(wx + b)

## 🛠️ Tech Stack
- Python 3.x
- PyTorch

## 🚀 How to Run
pip install -r requirements.txt
python train.py
