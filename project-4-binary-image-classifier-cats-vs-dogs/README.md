# Project 4 - Binary Image Classifier (Cats vs Dogs)

## 📌 Description
CNN-based binary classifier to distinguish cats from dogs.

## 🎯 Goal
Classify images as Cat or Dog using CNN.

## 📁 Data Structure
data/train/cat/ ← cat images
data/train/dog/ ← dog images

## 🛠️ Tech Stack
- Python 3.x
- PyTorch
- torchvision
- Pillow

## 🚀 How to Run
pip install -r requirements.txt
python train.py
python predict.py
