## 📌 Description
Simplified VGG-16 CNN trained on CIFAR-10 dataset.

## 🎯 Architecture
13 Conv layers + 2 FC layers with MaxPooling and Dropout.

## ⚠️ Note
Using 64x64 input for CPU compatibility.

## 🛠️ Tech Stack
- Python 3.x
- PyTorch
- torchvision

## 🚀 How to Run
pip install -r requirements.txt
python train.py
python predict.py
